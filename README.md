# 🌍 Travel Tracker

A sleek, interactive web application to track and visualize the countries you've visited.

## 🚀 Features
* **Interactive Map**: Uses `jsVectorMap` to highlight visited countries in real-time.
* **Smart Search**: Search for countries by name; the app automatically handles case-sensitivity and partial matches.
* **Persistent Storage**: Powered by PostgreSQL to keep your travel history safe.
* **Dynamic UI**: Modern dark-mode interface with a "tag cloud" of visited country names.
* **Error Handling**: Prevents duplicate entries and notifies users of unrecognized country names.

## 🛠️ Tech Stack
* **Frontend**: EJS (Embedded JavaScript templates), CSS3, JavaScript.
* **Backend**: Node.js, Express.js.
* **Database**: PostgreSQL.
* **Libraries**: `pg` (Postgres client), `body-parser`, `jsVectorMap`.
